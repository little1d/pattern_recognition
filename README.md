# 模式识别课程作业

感谢唐旭老师，大家没有往年题也不用害怕，期末考试很简单😁

## 项目结构

- **assets/** - 实验报告和结果图片
  - Fisher 判别分析实验报告.pdf
  - kmeans_vs._FCM.pdf
  - KNN.pdf
  - SVM 算法.png

- **Fisher/** - Fisher 判别分析算法
- **KNN/** - K近邻算法
- **SVM/** - 支持向量机算法
- **kmeans-vs-fcm/** - K-means 与 FCM 聚类算法对比
- **dpca/** - 密度峰值聚类算法

## 数据集

- Iris（鸢尾花数据集）
- Sonar（声纳数据集）
- MNIST（手写数字数据集）

## 运行方式

每个算法目录下都有相应的主程序文件（`main.py`）和测试文件（`test.ipynb`），可以直接运行查看结果。

## 详细报告

详细的实验报告和分析结果请查看 `assets/` 目录下的 PDF 文件。
